Front Door — Authentication

Sign Up with Email, Password & Telegram ID

Users fetch their Telegram User ID via the system-owned Auth Bot (/start). On submit, an OTP is sent to their Telegram. The UI auto-switches to OTP entry and completes login on success.

Control Center — Dashboard

Add Bots & Upload Users (CSV/JSON/TXT)

Paste a bot token to validate and add it. Upload user files with a live scanner that extracts unique numeric IDs, de-duplicates across uploads, and updates counts per bot in real time.

Main Event — Broadcasting

Select Bots (all pre-selected) & Send

Compose in Markdown and broadcast instantly. Watch the live status panel update every few seconds: Sent, Failed (blocked users), and Total Recipients until Complete.
